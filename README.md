# isSHA1

Check to see if something is SHA1 

## Install

    $ npm install --save issha1

## Usage

    const issha1 = require('issha1');

    issha1.test('a94a8fe5ccb19ba61c4c0873d391e987982fbbd3');
    issha1.test('NOTSHA1');
